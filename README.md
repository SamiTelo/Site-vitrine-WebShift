## Installation & Launch

### Clone the project

```bash
git clone https://github.com/samiTelo/nom-du-site.git
cd nom-du-site
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
